Ejercicio 11

Escriba instrucciones que asignen enteros aleatorios a la variable n en los siguientes rangos
a) 1 ≤ n ≤ 2
b) 1 ≤ n ≤ 100
c) 0 ≤ n ≤ 9
d) 1000 ≤ n ≤ 1112
e) –1 ≤ n ≤ 1
f ) –3 ≤ n ≤ 11

Respuesta:
a) var NumR=Math.floor(Math.random()*2);
b) var NumR=Math.floor(Math.random()*101);
c) var NumR=Math.floor(Math.random()*10);
d) var NumR=Math.floor(Math.random()*(1112-1000)+1000);
e) var NumR=Math.floor(Math.random()*(1-(-1))-1);
f) var NumR=Math.floor(Math.random()*(11-(-3))-3);
