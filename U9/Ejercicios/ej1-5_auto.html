9.0 Ejercicios de Autoevaluación

9.1 Completa Complete las siguientes oraciones:

a) Los módulos de programa en JavaScript se llaman __________.
b) Una función se invoca mediante el uso de un(a) __________.
c) Una variable que se conoce sólo dentro de la función en la que se define se llama __________.
d) La instrucción __________ en una función invocada puede usarse para pasar el valor de una expresión de regreso a la función que hizo la llamada.
e) La palabra clave __________ indica el comienzo de la definición de una función.

Respuesta:
a) funciones.
b) llamada a función.
c) variable local.
d) return.
e) function.

9.3 Complete las siguientes oraciones:
a) Las funciones definidas por el programador, variables globales y funciones globales de JavaScript son todas partes del objeto __________.
b) La función __________ determina si su argumento es un número o no.
c) La función __________ recibe un argumento de cadena y devuelve una cadena en la que todos los espacios, puntuación, caracteres de acentuación y cualquier otro carácter que no esté en el conjunto de caracteres ASCII se codifican en un formato hexadecimal.
d) La función __________ recibe un argumento de cadena que representa el código de JavaScript a ejecutar.
e) La función __________ recibe una cadena como su argumento y devuelve una cadena en donde se decodifican todos los caracteres que se codificaron antes con escape.

Respuestas:
a) Global.
b) isNan.
c) escape.
d) eval.
e) unescape.

9.5 Localice el error en cada uno de los siguientes segmentos de programa y explique cómo corregirlo:
a) method g() { document.writeln( "Dentro del método g" );}
b) // Esta función debe devolver la suma de sus argumentos function suma(x, y) { var resultado; resultado = x + y; }
c) function f( a ) { document.writeln( a ); }

a) Error: method no es la palabra clave que se utiliza para comenzar la definición de una función. Corrección: cambiar method a function.
b) Error: se supone que la función debe devolver un valor, pero no es así. Corrección: eliminar la variable resultado y colocar la instrucción return x + y; en la función o agregar la siguiente instrucción al final del cuerpo de la función: return resultado;
c) Error: el punto y coma después del paréntesis derecho que encierra la lista de parámetros. Corrección: elimine el punto y coma después del paréntesis derecho de la lista de parámetros.
