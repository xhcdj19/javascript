Ejercicio 16. u9

Escriba segmentos de programa que realicen cada una de las siguientes tareas:
a) Calcule la parte entera del cociente cuando el entero a se divide entre el entero b.
b) Calcule el residuo entero cuando el entero a se divide entre el entero b.
c) Use las piezas de programa que se desarrollaron en las partes (a) y (b) para escribir una función llamada mostrar- Digitos que reciba un entero entre 1 y 99999, para luego imprimirlo como una serie de dígitos, de los cuales cada par se separa mediante dos espacios. Por ejemplo, el entero 4562 debería imprimirse como 4 5 6 2
d) Incorpore la función desarrollada en la parte (c) en una secuencia de comandos que reciba un entero de un diá-logo prompt e invoque mostrarDigitos para pasar a la función el entero introducido.
