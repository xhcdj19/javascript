Dada y = ax^3 + 7, ¿cuáles de bs siguientes instrucciones de JavaScript son correctas para esta ecuación?

a ) y = a * x * x * x + 7;
b) y = a * x * x * (x + 7);
c) y = (a * x) * x * (x + 7);
d) y = (a * x) * x * x + 7;
e) y = a* (x * x * x ) + 7 ;
f) y = a * x * (x * x + 7);

Respuesta:
